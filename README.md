# Projek-Python-Pro-Hanafi

## Judul Projek
Projek ini berjudul **polucount.bot**

## Deskripsi
polucount.bot adalah bot yang dapat menghitung karbon yang dihasilkan oleh kendaraan yang dipilih user beserta jaraknya. Adapun jenis kendaraan yang dapat diketahui jumlah karbonnya yaitu
* Mobil
* Motor
* Bus
* Kereta Api
* Pesawat
* Kapal Laut

## Teknologi
Projek ini menggunakan `python 3.11`. Library yang digunakan antara lain:
* `discord.py`
* `numpy`
* `scipy`
* `tensorflow`
* `teachable machine`
